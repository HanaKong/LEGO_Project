# 구현기능

- 회원 관련
  - 로그인, 로그아웃, 회원가입, 회원탈퇴
  - ID / 비밀번호 찾기( 임시 비밀번호 ) - 이메일 발송
  - 마이페이지 ( 회원정보 수정, 비밀번호 수정, 구매내역, 관심 목록, 회원탈퇴 )
  - VIP 등록 ( 회원 등급 )

- 관리자 페이지
  - 회원 관리
  - 상품 관리
  - 스토어 지도 관리
  - Q&A 답변
 
- 상품 페이지
  - 가격순 정렬, 카테고리별 정렬
  - 장바구니 추가
 
- 상품 상세
  - 관심 상품 추가
  - QnA 등록
  - 장바구니 담기
  - 상품 리뷰 확인
  - 상세정보, 리뷰, Q&A 버튼 클릭시 스크롤 이동

- 장바구니
  - 선택 삭제
  - 선택 주문

- 결제
  - 포인트 사용 ( 직접 입력 + 전액 사용 버튼 )
  - 다음 주소 API를 통한 배송 정보 수정
  - 주문자 정보 수정
  - 카카오 페이 API를 통한 결제
 
- 고객지원
  - 레고 스토어 찾기 ( 카카오 지도 API 사용 )

- 로그인
